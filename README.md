## 👨‍💻 Sobre mim
 
Tenho 19 anos, sou de Goiás e moro em Ceilândia-DF. Estou indo para o **4º semestre de Ciência da Computação** na Universidade Católica de Brasília (UCB) e tenho foco em desenvolvimento web — tanto no frontend quanto no backend.
 
Gosto de construir projetos reais para consolidar o que aprendo: já desenvolvi desde sites estáticos até sistemas fullstack com autenticação JWT, controle de permissões por perfil e integração com banco de dados.
 
- 🎯 Foco atual: **React**, **Node.js** e **TypeScript**
- 📚 Aprendendo: estruturas de dados, arquitetura de APIs REST e boas práticas de código
- 🤝 Aberto a colaborações e projetos em grupo
